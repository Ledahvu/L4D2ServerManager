## 🇻🇳 Tiếng Việt

**L4D2 Ultimate Manager** là giải pháp "All-in-One" tối thượng dành cho các Server Admin và người chơi Left 4 Dead 2. Quên đi việc phải gõ lệnh SteamCMD thủ công, giải nén nhầm thư mục hay hoa mắt vì hàng tá mã ID Workshop. Công cụ này sẽ tự động hóa mọi thứ!

### ✨ Tính năng nổi bật
* 🚀 **Quản lý Server 1-Click:** Tự động gọi SteamCMD để cài đặt, cập nhật và verify dữ liệu L4D2 Server. Quản lý đa Profile độc lập.
* 🧩 **Hệ thống Cài đặt Extension Thông minh:** Tự động tìm link tải mới nhất, vượt tường lửa và cài đặt MetaMod, SourceMod. Bao gồm thuật toán "Phẫu thuật ZIP" cân bằng đường dẫn tự động cho **L4DToolZ**.
* 🗺️ **Bậc thầy Custom Map:** Dán link Workshop -> Tự động tải -> Bóc tách file `.vpk` lấy map `.bsp` -> Tự động gọi Steam API để hiển thị **Tên thật của Map** siêu đẹp.
* 🌐 **Global Server Browser:** Quét server toàn cầu với API của Steam. Tích hợp bộ lọc **Anti-Fake Region** (Tự động đá các server cố tình fake ping hiển thị sai khu vực) và chuẩn hóa tên Quốc gia.
* 🎨 **Giao diện Tùy chỉnh:** Hỗ trợ 3 bộ Theme (Dark, Light, Hacker Matrix) với OwnerDraw ListBox hiển thị đa luồng thông tin.

### 🚀 Cài đặt & Sử dụng
1. Tải [Bản phát hành mới nhất (Latest Release)](#).
2. Chạy file `L4D2UltimateManager.exe`.
3. Chọn thư mục lưu trữ Profile và bắt đầu tạo Server của riêng bạn!

## 🇬🇧 English

**L4D2 Ultimate Manager** is the ultimate All-in-One solution for Left 4 Dead 2 Server Admins and players. Forget about manually typing SteamCMD commands, extracting files to the wrong directories, or getting confused by dozens of Workshop ID codes. This tool automates everything!

### ✨ Key Features
* 🚀 **1-Click Server Management:** Automatically invokes SteamCMD to install, update, and verify L4D2 Server data. Supports multi-profile management.
* 🧩 **Smart Extension Installer:** Automatically fetches the latest download links, bypasses simple firewalls, and installs MetaMod and SourceMod. Includes a "ZIP Surgery" algorithm to auto-balance paths for **L4DToolZ**.
* 🗺️ **Custom Map Master:** Paste Workshop links -> Auto-download -> Extract `.vpk` to find `.bsp` maps -> Automatically calls Steam API to display the **Real Map Names** beautifully.
* 🌐 **Global Server Browser:** Scans global servers using the Steam API. Built-in **Anti-Fake Region** filter (automatically kicks out servers faking their ping/location) and Country name normalization.
* 🎨 **Custom UI:** Supports 3 Themes (Dark, Light, Hacker Matrix) with an OwnerDraw ListBox for multi-line information display.

### 🚀 Installation & Usage
1. Download the [Latest Release](#).
2. Run `L4D2UltimateManager.exe`.
3. Select your Profile storage directory and start building your server!

## 🇨🇳 中文

**L4D2 终极管理器** 是为《求生之路 2》服务器管理员和玩家打造的终极“多合一”解决方案。忘记手动输入 SteamCMD 命令、错误解压文件或被几十个创意工坊 ID 代码弄晕的烦恼吧。这个工具能将一切自动化！

### ✨ 核心功能
* 🚀 **一键服务器管理：** 自动调用 SteamCMD 来安装、更新和验证 L4D2 服务器数据。支持独立多配置文件管理。
* 🧩 **智能插件安装器：** 自动获取最新下载链接并安装 MetaMod 和 SourceMod。包含一个“ZIP 手术”算法，自动平衡 **L4DToolZ** 的路径。
* 🗺️ **自定义地图大师：** 粘贴创意工坊链接 -> 自动下载 -> 提取 `.vpk` 找到 `.bsp` 地图 -> 自动调用 Steam API 显示**真实的地图名称**。
* 🌐 **全球服务器浏览器：** 使用 Steam API 扫描全球服务器。内置 **防伪造区域 (Anti-Fake Region)** 过滤器（自动剔除伪造延迟/位置的服务器），并实现国家名称标准化。

### 🚀 安装与使用
1. 下载 [最新版本 (Latest Release)](#)。
2. 运行 `L4D2UltimateManager.exe`。
